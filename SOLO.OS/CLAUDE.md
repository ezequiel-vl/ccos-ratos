# SOLO.OS

Projeto e curso do Ezequiel. Ensina freelancers e agências a construir um sistema operacional pro próprio negócio usando o Claude Code como motor.

## Contexto

No início de toda conversa, ler `../_contexto/empresa.md` e `../_contexto/estrategia.md` pra entender onde o Ezequiel está no negócio.

O SOLO.OS é um projeto do Ezequiel, não um cliente. O tom aqui é o dele — informal, direto, primeira pessoa. Ler `../_contexto/preferencias.md` pra restrições de escrita.

## O que fica aqui

- Materiais do curso (aulas, roteiros, scripts)
- Conteúdo produzido sobre o método SOLO.OS
- Templates ensinados no curso
- Documentação do sistema

## Estrutura sugerida

```
SOLO.OS/
  CLAUDE.md           ← este arquivo
  aulas/              ← roteiros e materiais por módulo
  conteudo/           ← posts, carrosseis, newsletters sobre o curso
  templates/          ← templates que são entregues no curso
  landing/            ← página de vendas e materiais de captação
```

## Regras

- Conteúdo aqui fala com freelancers e agências — público diferente dos clientes da ezeOS
- O método ensinado é baseado no que o Ezequiel usa na própria operação
- Nomear arquivos com data ou número de módulo: `modulo-01-setup.md`, `2026-06-13-post-instagram.md`
